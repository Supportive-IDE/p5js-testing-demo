Instructions

Create a canvas at a size of your choosing (at least 600 x 400 is recommended) and use p5.js to draw a character. Then, make your character follow the mouse as it moves.

To make a character move with the mouse, you will need to specify the coordinates of each shape as calculations relative to the mouse. It will be easier to design a static character first, rather than try to make it move with the mouse right away:

  • Create two variables and use them to store the centre point of your canvas on each axis. For example, if your canvas is 600 by 400, you would create the variables as follows: let centreX = 300; let centreY = 200;
  • Design your character and specify each coordinate of each shape as a calculation relative to the appropriate variable. For example, an x coordinate of 200 can be expressed as centreX - 100.

Finally, replace the two variables created earlier with the built-in variables mouseX and mouseY.
